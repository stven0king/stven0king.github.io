总结

事件由 dart 到 C 再到 Java ，相应由 Java 到 C 再到 dart 的过程可以简单用一下步骤叙述：

1、Application启动的时候加载flutter的so文件；
2、在加载so的时候注册了一系列的相关平台的函数以及操作类；
3、dart调用C层的方法顺便将数据传递给C层；
4、C层调用相关平台的注册的类的对应方法，
5、对应平台进行数据处理并返回数据；
6、事件到达系统底层之后找到事件的相应的句柄进行回调；
