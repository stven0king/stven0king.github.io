将当前目录切换到Python的worspace下，输入新建的项目名称：

  • init.py：Python特性，可以是空文件，表明这个文件夹是一个可以导入的包。
  • settings.py：配置文件，本文主要修改数据库信息、模板目录、加载模块的信息。
  • url.py：URL配置文件，指定函数与URL的映射关系。
  • wsgi.py：本文中不会使用，nginx/apache+wsgi在生产环境中运行Django时使用
